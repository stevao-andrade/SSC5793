Projeto 2 - Reuso de software.

Alunos: Stevão & Marcelo

Contém 2 projetos:

ExamsRepository - Possui um meta modelo de referencia para um sistema que representa um repositorio de questões.

DocumentsGenerator - Projeto em Accelo, que usa uma instancia do modelo projetado no projeto ExamRepository.
				   - Implementa 3 soluções:
				   							- Gerador de provas com as questões instanciadas no repositorio criado com base no metamodelo
				   							- Gerador de gabarito para as provas instanciadas a partir dos repositorio criado
				   							- gerador de codigo fonte em Python que possibilita a execução das provas existentes dentro da instancia do metamodelo.

Criado utilizando a IDE Eclipse Model (Oxygen)

Para a perfeita utilização, é necessario a instalação dos plugins ferentes a modelos disponiveis no Eclipse market (Help -> Install New Software - Modelling Tools).
